Fonctionnalités

Tout ce dont vous avez besoin

Six modules pensés pour transformer votre relation avec la cuisine quotidienne.

Bibliothèque de recettes

Recherche avancée, filtres par saison, durée et catégorie. Scroll infini. Édition complète avec ingrédients et instructions.

Planning de repas

Calendrier glissant 3/5/7 jours. Ajoutez déjeuners et dîners d'un clic. Navigation fluide avec prefetch intelligent.

Liste de courses intelligente

Import automatique des ingrédients depuis vos recettes. Liste "Habituels" séparée. Groupement par rayon, mémorisation des labels.

Suggestions IA

Nouveau

Assistant conversationnel embarqué. Suggestions personnalisées selon vos stocks, saison et historique. Compatible Anthropic, OpenAI, Google et Ollama.

Statistiques & insights

Top recettes, top ingrédients, streak de planification, taux de restes, couverture du catalogue. Périodes 30j / 90j / 12m.

Thème personnalisable

Dark / light / système. Couleur d'accent au choix parmi 8 palettes oklch soigneusement sélectionnées. Interface qui vous ressemble.

Installation

En route en 5 minutes

Bonap se connecte à votre instance Mealie existante. Aucune migration, aucune perte de données.

1

Installer Mealie

Déployez Mealie sur votre serveur avec Docker. Une instance existante fonctionne directement.

# docker-compose.yml
docker compose up -d
2

Lancer Bonap

Clonez le repo ou utilisez l'addon Home Assistant inclus dans le dépôt.

# ou via npm
git clone && npm run dev
3

Configurer

Renseignez l'URL de votre Mealie et votre token API dans le fichier .env.

VITE_MEALIE_URL=...
VITE_MEALIE_TOKEN=...

Profitez

Toutes vos recettes Mealie sont disponibles immédiatement dans la nouvelle interface.

Vos recettes, planning et courses — au même endroit, plus agréable a utiliser.

Addon Home Assistant inclus

Un addon HA est fourni dans le dépôt pour une intégration native dans votre instance Home Assistant.
